# 🎓 Student Dropout Prediction System

A Machine Learning project that predicts whether a student is at risk of dropping out using academic, social, and behavioral attributes.
The system uses Logistic Regression and Decision Tree algorithms to identify students who may need early intervention.

# 📌 Project Overview

Student dropout is a major issue in educational institutions. Early identification of at-risk students allows universities and schools to take preventive measures.
This project builds a predictive machine learning model that analyzes student data and estimates the probability of dropout.

The project demonstrates:
Data preprocessing
Feature engineering
Logistic Regression for probability prediction
Decision Tree for explainable rules
Model evaluation and visualization

# 🎯 Objectives

Predict whether a student is likely to drop out
Identify the key factors influencing dropout
Provide interpretable decision rules using Decision Trees
Compare performance of two machine learning models

# 📂 Dataset

Dataset used: UCI Student Performance Dataset

It contains student academic and social information such as:
Study time
Absences
Family background
Internet access
Previous grades
Alcohol consumption
School support

Target variable created:

Dropout Risk
0 → Low Risk
1 → High Risk

Students with final grade < 10 are considered at risk.
